Installation

Stable release

To install heatpro, run this command in your terminal:

$ pip install heatpro

This is the preferred method to install heatpro, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources

The sources for heatpro can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/CEA-Liten/heatpro

Once you have a copy of the source, you can install it with poetry:

$ poetry install <path_to_repository>
